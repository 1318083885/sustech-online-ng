# ❓站点帮助

## 添加南科手册到桌面

南科手册支持[PWA](https://sspai.com/post/60928)[(Progressive Web App)](https://web.dev/progressive-web-apps/)，这意味着您可以将它添加到桌面，并作为一个本地应用使用。因此，在下载应用后，即时在网络不佳的环境下，您依然可以阅读南科手册上的内容。

### 如何添加

以Chrome为例：

![三个点](./pwa/add1.png)

1. 点击右上角的三个点

![点击“添加到主屏幕”](./pwa/add2.png)

2. 点击“添加到主屏幕”

![点击“添加”](./pwa/add3.png)

3. 点击“添加”
